Descripción

Este proyecto fue creado para un cliente el cual solicitó un sitio web basado en el diseño del mismo con la funcionalidad de sincronizar mensajes entre 2 o más dispositivos, posibilidad de búsqueda entre los mensajes y un panel de administración.

Las contraseñas y los mensajes se guardan de forma encriptada y siempre se sanitiza el texto para evitar ataques XSS. El panel de administrador permite decidir si un usuario tiene permiso o no para usar el sitio web y es posible que se le agreguen más funciones en el futuro.
